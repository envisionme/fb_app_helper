***Installation***
1. install module
2. go to admin/settings/fbconnect/helper
3. click activate checkbox and save

***Usage***
1. add your own specific javascript functions to js/canvas.js and 
call the functions in attachCanvasHelpers()
2. add css to css/canvas.css that you want specifically for your canvas app
